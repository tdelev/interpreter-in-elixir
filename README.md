# Interpreter in Elixir

This is implementation of the well-known [Writing and Interpreter in Go](https://interpreterbook.com/) book in Elixir.

## Running the interpreterbook

```bash
$ cd monkey/
$ mix run repl.exs
```

## Running the tests

```bash
$ cd monkey/
$ mix test
```
